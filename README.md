## **MucicalWorld - MERN stack application with social elements**

### Description
This website is my project for my resume, this is a MERN applicaiton that <ins>**focuses on security while showcasing social elements**</ins> achievable with MERN stack, <ins> **no css library or ai was used for css code all the css code is raw and hand written**</ins>
>Account creation is disabled, <ins>username and password provided in resume</ins>, to get resume or username and password please contact me

### Highlights
1. All the css was written by hand no library or ai was used
2. Implementation of security practices
   - Password hashing (Bcrypt)
   - Nosql injection prevention
   - Protected Routes

#### This projects demonstrates:
1. Robust client and server side user authentication
2. Optimistic UI for realtime feedback
3. Secure storage of passwords using bcrypt
4. Resisting nosql injection attacks
5. JWT tokens in httponly cookie
6. Loading states and graular loading states
7. Monolithic architecture
8. Protected routes and secret environment variables
9. Multi format upload pipeline and MIME type filtering on fontend and backend
10. Autologin using JWT
11. Dynamic routing(user's page) font end get's data based on url paramaters
12. Post feed based on filters
13. Password updation require meeting lenght, number, symbol and space requirement
14. Updation of profile picture with MIME type filtering on fontend and backend
